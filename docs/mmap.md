# Boot and Kernel Memory Map


## Physical Memory
------------------------------------------------

0x1000 --- Stage 2 Bootloader
...
0x74FB --- int pt_cnt (number of pages taken by the pagetables at 0x100000)
0x7500 --- int bios_mmap_count
0x7504 --- struct bios_mmap_count bios_mmap[?] (size depends on number of entries)
...
0x7C00 --- Boot loader stack top
0x7C00 --- boot sector
...
...
0x14FF0 ---  Kernel stack top
0x15000 --- Stage 3 Bootloader
...
...
0x20000 --- Kernel
...
...
...
0x100000 --- struct pagetable master_pml[?] (size depends on ram size)

--------------------------------------------------


## Virtual Memory
--------------------------------------------------

0x0000000000000000 --- User Address Space
...
...
...
0xFFFF800000000000 --- Start of Kernel Address Space
0xFFFF800000000000 --- Start of full RAM map for full kernel access to RAM
...
...
...
0xFFFFFFFF80000000 --- Start of virtual Kernel

---------------------------------------------------
